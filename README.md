##### 简述

用于计算合租时各种麻烦的计算问题的工具，目前已有功能：  
* 食材计算

##### 已知问题：
* 已修复 hashMap.get返回null
  * 错误原因：用户名重复或者不足时会尝试获取大于size的用户
  * 解决方式：调用get前使用if检查是否能够获取数据
* 已修复 删除用户后id断层
  * 错误原因：MySQL特性
  * 解决方式：每次写入新的用户时重写一遍user表
